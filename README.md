# StudyDeck

StudyDeck is a lightweight, browser-based flashcard app for reviewing programming questions and answers.  
It uses JSON files to store questions, making it easy to update and expand your study material.  

## Features

- **Multiple Decks:** Users can select from multiple study decks (e.g., C#, JavaScript).  
- **Clickable Cards:** Click on a card to reveal the answer.  
- **Search Functionality:** Quickly search questions inside the selected deck.  
- **Dark/Light Mode Toggle:** Comfortable studying in different lighting conditions.  
- **Responsive Header Layout:** 
  - Back button appears in the upper-left when viewing a deck.  
  - Dark/Light mode toggle appears in the upper-right.  
  - Deck title stays centered.  
- **Fully Client-Side:** No backend required; works entirely in the browser.  
- **Future Expandability:** Easily add/delete cards or create new decks.
